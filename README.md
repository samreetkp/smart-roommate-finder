# RoomAi

Starter web app for a roommate matching platform with a clean React + TypeScript setup.

## Features in this starter

- Search by name, hobbies, or bio
- Filter by city
- Pet-friendly filter
- Compatibility score sorting (mock AI-style scoring)

## Run locally

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - start development server
- `npm run build` - type-check + production build
- `npm run preview` - preview production build
- `npm run lint` - run ESLint
